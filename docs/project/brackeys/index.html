Brackeys GameJam

Barckeys Game Jam

2025.1

This will be KBVE entry to the Barckeys Game Jam!! For this entry, we will be using Godot with Rust

The theme for this jam is ERASE and the wild cards are:

  • Mimic - Something is not what it seems.
  • Style Shift - Visuals switch between two art styles.
  • Shopaholic - include some kind of shopping mechanism.

GameDocs

Game description You wake up in a miner spaceship, and everybody is gone. And worse of it all, sparks appear in different parts of the ship, concluding that the space ship already undergoes different hardship and you hope that nothing goes wrong.

UIUX

An analog spaceship UI could have scratched metal textures, dimly glowing buttons, slightly flickering holographic elements, and a worn-down control panel vibe. crt display? inspired by spaceteam

Gameplay Loop

  • Mine asteroids and harvest scraps across the space to earn money.
  • Fix and upgrade you ship.
  • You will encounter various anomly that can be good or bad for you.
